# BasketOps AI

### Basketball Analytics, Shot Intelligence, AI & MLOps Platform

BasketOps AI è una piattaforma end-to-end avanzata per l'analisi dei dati di pallacanestro, la visualizzazione intelligente dei tiri (Shot Charts), la predizione delle performance tramite Machine Learning e il monitoraggio dei modelli in produzione secondo le migliori pratiche MLOps e DevOps.

## 🚀 Funzionalità Principali
- **Shot Chart & Heatmaps Interattive:** Visualizzazione spaziale dei tiri realizzati e sbagliati.
- **AI Shot Success Predictor:** Calcolo in tempo reale della probabilità di realizzazione di un tiro.
- **Player Clustering & Similarity Search:** Identificazione automatica degli archetipi dei giocatori (es. *Stretch Big*, *Rim Finisher*) e ricerca di profili simili.
- **MLOps Architecture:** Tracking degli esperimenti e versionamento dei modelli con MLflow.
- **Enterprise DevOps:** Pipeline CI/CD complete, containerizzazione (Docker) e orchestrazione (Kubernetes).
- **Osservabilità:** Monitoraggio delle metriche applicative e di data drift con Prometheus e Grafana.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Backend:** FastAPI, Python
- **Database:** PostgreSQL
- **AI/ML:** Scikit-Learn, XGBoost/LightGBM, MLflow
- **DevOps/MLOps:** Docker, Kubernetes, GitHub Actions, Terraform
- **Monitoring:** Prometheus, Grafana
