# BasketOps

BasketOps AI è una piattaforma che permette di:

* analizzare dati di basket
* visualizzare Shot Charts
* generare heatmap
* predire probabilità di realizzazione dei tiri
* identificare archetipi di giocatori
* monitorare modelli ML
* automatizzare training e deployment
* applicare pratiche DevOps e MLOps
